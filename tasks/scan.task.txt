1. Rate limit check (ScanRateLimit by IP)         → return 429 if exceeded
2. Hash the incoming token with HMAC + secret
3. Look up token_hash in DB
4. If not found          → log ScanResult.INVALID,      return generic "not found"
5. If REVOKED            → log ScanResult.REVOKED,      return "card inactive"
6. If EXPIRED            → log ScanResult.EXPIRED,      return "card expired"
7. If INACTIVE           → log ScanResult.INACTIVE,     return "card inactive"
8. If ACTIVE:
   a. Check LocationConsent → capture lat/lng if enabled
   b. Apply ProfileVisibility filter
   c. Write ScanLog with full device/IP/location data
   d. Trigger anomaly detection (async, don't block response)
   e. Send SCAN_ALERT notification to primary parent (async)
   f. Return filtered emergency profile
9. All paths must respond in constant time (pad with sleep if needed)
